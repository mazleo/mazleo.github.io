RU Vacant

RU Vacant is the first android application I have worked on that I plan to release into the Play Store. I read most of the guides on the documentation that I needed in this application, from essentials such as the Activity Lifecycle and Intents to less essential topics like Animated Vector Drawables. Although I wish I started and finished this application before I graduated college because it would have been a really useful application for me, my goal is to learn more about android development, and eventually master it in the future. I will continue working on this application, probably continuously refactoring the code to improve it as I go. Additional goals while working on this application is to learn more about proper architecture design, and learn more about important concepts such as what makes code maintainable.
